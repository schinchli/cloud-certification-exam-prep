## Exam Format (ACE — Representative)

| Attribute | Details |
|---|---|
| Questions | 50–60 multiple choice and multiple select |
| Duration | 2 hours |
| Delivery | Remote or in-person (Kryterion) |
| Passing Score | ~70% (not published officially) |
| Cost | $200 USD |

*Always verify current format at [cloud.google.com/learn/certification](https://cloud.google.com/learn/certification).*
