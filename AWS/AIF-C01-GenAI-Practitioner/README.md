# AWS Certified AI Practitioner

This folder contains comprehensive study materials for the AWS Certified AI Practitioner (AIF-C01) exam.

## Contents

- `AWS_GenAI_Practitioner_Study_Guide.md` - Complete study guide with all key concepts and services

## Exam Overview

The AWS Certified AI Practitioner validates foundational knowledge of artificial intelligence (AI) and machine learning (ML) concepts and AWS AI/ML services.

### Domain Breakdown
1. **Fundamentals of AI and ML (20%)** - Basic AI/ML concepts, terminology, applications
2. **Fundamentals of Generative AI (24%)** - GenAI concepts, foundation models, prompt engineering
3. **Applications of Foundation Models (28%)** - Model selection, customization, inference
4. **Guidelines for Responsible AI (14%)** - Bias, fairness, explainability, security
5. **Security, Compliance, and Governance for AI Solutions (14%)** - AWS security services, governance frameworks

### Key Services to Master

#### Core AI/ML Services
- **Amazon Bedrock** - Foundation model service with guardrails
- **Amazon SageMaker** - Complete ML platform and ecosystem
- **Amazon Rekognition** - Image/video analysis
- **Amazon Comprehend** - Natural language processing
- **Amazon Textract** - Document analysis
- **Amazon Transcribe** - Speech-to-text
- **Amazon Polly** - Text-to-speech

#### Specialized Services
- **Amazon Q Business/Developer** - AI assistants
- **Amazon Kendra** - Intelligent search
- **Amazon Personalize** - Recommendation engine
- **Amazon Forecast** - Time series forecasting

#### Amazon Nova Models
- **Understanding Models**: Micro, Lite, Pro, Premier
- **Creative Models**: Canvas (image), Reels (video)
- **Speech Model**: Sonic (speech-to-speech)

### Exam Details
- **Duration**: 90 minutes
- **Questions**: 65 multiple choice/multiple response
- **Passing Score**: 700/1000
- **Cost**: $150 USD
- **Validity**: 3 years

## Key Focus Areas

### Must-Know Concepts
- **RAG vs Fine-tuning** - When to use each approach
- **Prompt Engineering** - Chain of thought, negative prompting
- **Model Parameters** - Temperature, Top-K, Top-P
- **Evaluation Metrics** - Accuracy, precision, recall, F1, BLEU, ROUGE
- **Security Threats** - Prompt injection, jailbreaking, hallucinations

### Service Differentiators
- **SageMaker components** - Know what each tool does
- **Bedrock vs SageMaker** - When to use foundation models vs custom models
- **Vector databases** - OpenSearch vs DocumentDB capabilities
- **Inference options** - Real-time vs batch vs serverless

### Responsible AI
- **Bias detection** - SageMaker Clarify capabilities
- **Explainability** - SHAP, feature attribution
- **Security frameworks** - GenAI Security Scoping Matrix
- **Governance** - Model cards, audit trails

## Success Tips

1. **Hands-on Practice**: Use AWS Free Tier to experiment with services
2. **Understand Use Cases**: Know when to use each service/approach
3. **Focus on Integration**: How services work together in AI workflows
4. **Security First**: Understand AI-specific security considerations
5. **Stay Current**: GenAI is rapidly evolving - focus on core concepts
