## Study Notes

### [AIF-C01 — AWS Certified AI Practitioner](./AIF-C01-GenAI-Practitioner/)
Foundational AI/ML certification covering AWS AI services and generative AI concepts.
Topics: Amazon Bedrock, SageMaker, Foundation models, Prompt engineering, RAG, Fine-tuning

### [MLS-C01 — AWS Certified Machine Learning – Specialty](./MLS-C01-ML-Specialty/)
Advanced ML certification covering data engineering, modeling, and ML operations.
Topics: Feature engineering, model training, SageMaker pipelines, MLOps, evaluation

### [SCS-C02 — AWS Certified Security – Specialty](./SCS-C02-Security-Specialty/)
Security-focused certification covering IAM, threat detection, data protection, and compliance.
Topics: IAM advanced policies, GuardDuty, Security Hub, KMS, Shield, WAF, compliance

Comprehensive professional-level notes across 16 topic areas:
