# AWS Certified Machine Learning - Specialty

This folder contains comprehensive study materials for the AWS Certified Machine Learning - Specialty (MLS-C01) exam.

## Contents

- `AWS_ML_Associate_Study_Guide.md` - Complete study guide organized by exam domains

## Exam Overview

The AWS Certified Machine Learning - Specialty validates expertise in:

### Domain Breakdown
1. **Data Engineering (20%)** - Data repositories, transformation, ingestion
2. **Exploratory Data Analysis (24%)** - Data preparation, feature engineering, visualization  
3. **Modeling (36%)** - Algorithm selection, training, evaluation, hyperparameter tuning
4. **Machine Learning Implementation and Operations (20%)** - Deployment, monitoring, security

### Key Services to Master
- **Amazon SageMaker** (Core service - know all components)
- **Amazon Kinesis** (Data Streams, Firehose, Analytics)
- **AWS Glue** (ETL operations)
- **Amazon S3** (Data storage and lifecycle)
- **ML Services**: Rekognition, Comprehend, Textract, Polly, Transcribe, Personalize

### Exam Details
- **Duration**: 3 hours
- **Questions**: 65 multiple choice/multiple response
- **Passing Score**: 750/1000
- **Cost**: $300 USD
- **Validity**: 3 years

## Study Approach

1. **Foundation** (Week 1-2): Complete ML fundamentals course
2. **AWS Services** (Week 3-4): Deep dive into SageMaker and ML services
3. **Algorithms** (Week 5): Master built-in algorithms and use cases
4. **Practice** (Week 6): Intensive practice exams and hands-on labs
5. **Review** (Week 7): Final review and weak area reinforcement

## Success Tips

- **Hands-on Practice**: Use AWS Free Tier for practical experience
- **Algorithm Mastery**: Know when to use supervised vs unsupervised algorithms
- **Service Integration**: Understand how services work together in ML workflows
- **Evaluation Metrics**: Master precision, recall, F1-score, AUC calculations
- **Real-world Scenarios**: Focus on practical implementation challenges
