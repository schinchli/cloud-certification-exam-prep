## Sample Questions (AZ-900 Format)

**Q1.** Which cloud deployment model keeps all resources on-premises?

- A. Public cloud
- B. Hybrid cloud
- C. Private cloud ✅
- D. Community cloud

> **Why C:** Private cloud resources are hosted on-premises or in a dedicated environment not shared with other organizations.

---

**Q2.** What does high availability mean in the context of Azure?

- A. Protecting data through backups
- B. Ensuring a service remains accessible with minimal downtime ✅
- C. Distributing traffic across multiple regions
- D. Encrypting data at rest

> **Why B:** High availability refers to a system's ability to remain operational and accessible, typically measured by an SLA uptime percentage.

---

**Q3.** Which Azure service provides a managed relational database?

- A. Azure Blob Storage
- B. Azure Cosmos DB
- C. Azure SQL Database ✅
- D. Azure Table Storage

> **Why C:** Azure SQL Database is a fully managed relational database service. Cosmos DB is NoSQL; Blob Storage is object storage.
