## Sample Questions (ACE Format)

**Q1.** You need to deploy a containerized application with automatic scaling on GCP. Which service is most appropriate?

- A. Compute Engine
- B. Cloud Functions
- C. Google Kubernetes Engine (GKE) ✅
- D. App Engine Standard

> **Why C:** GKE provides managed Kubernetes for containerized workloads with built-in autoscaling.

---

**Q2.** Which GCP service provides globally distributed NoSQL storage with low latency?

- A. Cloud SQL
- B. Cloud Spanner
- C. Firestore ✅
- D. BigQuery

> **Why C:** Firestore is GCP's managed NoSQL database optimized for low-latency reads and writes at global scale.
