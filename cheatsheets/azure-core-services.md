# Azure Core Services Cheatsheet

> Quick reference for AZ-900, AI-900, and Azure certification exams.

---

## Compute

| Service | Use Case |
|---|---|
| Azure Virtual Machines | IaaS, full OS control |
| Azure App Service | PaaS web apps, APIs |
| Azure Functions | Serverless, event-driven |
| Azure Container Instances (ACI) | Run containers without orchestration |
| Azure Kubernetes Service (AKS) | Managed Kubernetes |

## Storage

| Service | Type | Use Case |
|---|---|---|
| Blob Storage | Object | Unstructured data, media, backups |
| Azure Files | File share | SMB/NFS shared storage |
| Azure Disk Storage | Block | VM persistent disks |
| Azure Data Lake Storage | Object (analytics) | Big data analytics |

## Database

| Service | Type | Use Case |
|---|---|---|
| Azure SQL Database | Relational | Managed SQL Server |
| Azure Cosmos DB | NoSQL (multi-model) | Global distribution, low latency |
| Azure Database for PostgreSQL | Relational | Managed PostgreSQL |
| Azure Cache for Redis | In-memory | Caching, session management |

## Networking

| Service | Use Case |
|---|---|
| Virtual Network (VNet) | Isolated network |
| Azure Load Balancer | Layer 4 load balancing |
| Application Gateway | Layer 7 load balancing + WAF |
| Azure CDN | Global content delivery |
| Azure DNS | Domain name management |
| VPN Gateway | Site-to-site and P2S VPN |

## Security & Identity

| Service | Use Case |
|---|---|
| Microsoft Entra ID (Azure AD) | Identity, SSO, MFA |
| Azure Key Vault | Secrets, keys, certificates |
| Microsoft Defender for Cloud | Security posture, threat protection |
| Azure Policy | Governance, compliance enforcement |
| Azure RBAC | Role-based access control |

## AI & ML

| Service | Use Case |
|---|---|
| Azure OpenAI Service | GPT-4, DALL-E, Whisper via API |
| Azure AI Services | Vision, Speech, Language, Translator |
| Azure Machine Learning | MLOps, model training and deployment |
| Azure AI Search | Cognitive search, vector search |
