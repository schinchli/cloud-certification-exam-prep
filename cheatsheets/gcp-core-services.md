# GCP Core Services Cheatsheet

> Quick reference for Associate Cloud Engineer and Professional Cloud Architect exams.

---

## Compute

| Service | Use Case |
|---|---|
| Compute Engine | IaaS virtual machines |
| App Engine | PaaS, auto-scaling web apps |
| Cloud Functions | Serverless, event-driven |
| Cloud Run | Serverless containers |
| Google Kubernetes Engine (GKE) | Managed Kubernetes |

## Storage

| Service | Type | Use Case |
|---|---|---|
| Cloud Storage | Object | Unstructured data, backups, static sites |
| Persistent Disk | Block | VM disks |
| Filestore | File (NFS) | Shared file storage |

## Database

| Service | Type | Use Case |
|---|---|---|
| Cloud SQL | Relational | MySQL, PostgreSQL, SQL Server |
| Cloud Spanner | Relational (global) | Globally distributed, strong consistency |
| Firestore | NoSQL | Document store, real-time apps |
| Bigtable | NoSQL (wide-column) | Time series, IoT, analytics |
| BigQuery | Data warehouse | Large-scale analytics SQL |
| Memorystore | In-memory | Redis/Memcached caching |

## Networking

| Service | Use Case |
|---|---|
| VPC | Virtual private network |
| Cloud Load Balancing | Global/regional load balancing |
| Cloud CDN | Content delivery network |
| Cloud DNS | Managed DNS |
| Cloud Armor | DDoS and WAF protection |

## Security & Identity

| Service | Use Case |
|---|---|
| Cloud IAM | Identity and access management |
| Cloud KMS | Key management |
| Secret Manager | Secrets storage |
| Security Command Center | Threat detection, compliance |

## AI & ML

| Service | Use Case |
|---|---|
| Vertex AI | Unified ML platform (train, deploy, MLOps) |
| Gemini API | Multimodal LLM API |
| Cloud Vision AI | Image recognition |
| Cloud Natural Language | Text analysis |
| Document AI | Document processing |

---

*Source: [Google Cloud Documentation](https://cloud.google.com/docs) · [Google Cloud Architecture Center](https://cloud.google.com/architecture)*
